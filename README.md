# koalaDrone

Firmware for the koalaDrone project.



#### Building

* Visual Studio 2015
* Visual Micro for VS 2015
* Arduino IDE
* [ESP8266 Arduino](https://github.com/esp8266/Arduino)

* Arduino settings:
	* Generic ESP8266 Module
	* Flash Size: 1M (64K SPIFFS)
